开环轨迹控制实现：小海龟走正五角星
开环轨迹控制的核心是预先规划好速度指令的时序，不依赖反馈，仅通过设定线速度、角速度和执行时间来控制小海龟的运动轨迹。下面以更具代表性的正五角星为例，实现完整的开环轨迹控制节点，并详细解释开环控制的设计思路。
实现思路
开环控制核心：无需传感器反馈，仅通过「线速度 + 执行时间」控制直行距离，「角速度 + 执行时间」控制转向角度。
正五角星几何参数：
内角：36°，外角（转向角度）：144°（正五角星每次转向需转 144°）；
每条边直行时间固定，转向时间根据角速度计算（转向时间 = 转向角度 (弧度) / 角速度）。
ROS 节点逻辑：初始化节点→创建发布者→循环发布「直行 + 转向」指令→异常处理 + 停止运动。
